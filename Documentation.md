# 0n1 UI Library
## Documentation
#### Getting Loadstring
```lua
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ThatGlitches/UILibrary/main/0n1Library"))()
```
#### Creating Window
```lua
local Window = Library:CreateWindow("Title")
```
#### Creating Tabs
```lua
local Tab = Window:CreateTab("TabName")
```
